# LocalTrades South Dundas

Prototype for a rural-first local services directory for South Dundas.

The app helps residents find local trades, contractors, and service providers who actually serve the surrounding rural area. It is currently a static prototype using a CSV seed database, Leaflet maps, category filtering, contractor profiles, quote/claim modals, and a guided first-glance tour.

## Local Preview

Run the local static server:

```bash
node server.mjs
```

Then open:

```text
http://localhost:8000/
```

## GitHub Pages

This prototype is static and can be deployed directly with GitHub Pages from the `main` branch.
